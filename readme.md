### Uso

**No Linux**

- Use `./start.sh` para rodar o projeto

**No Windows**
Rode os comandos:

1. `docker compose build`
2. `docker compose up`
3. `docker exec prodmais ./container_services_start.sh` para iniciar os serviços dentro do container.
